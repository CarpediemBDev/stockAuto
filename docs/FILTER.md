# 🛡️ High-Performance Market Scanner: Filter Logic

본 문서는 StockAuto 시스템이 수천 개의 미국 주식 중 매수 적기인 종목을 선별해내는 **3단계 필터링 파이프라인**을 정의합니다. 
단순한 지표를 넘어, 전문 단타 투자자들이 중요하게 여기는 핵심 요소를 수치화하여 종목을 선정합니다.

---

## 🏗️ 3-Step Filtering Pipeline

### 1단계: 씨앗 종목 발굴 (Seed Discovery)
전체 시장에서 유동성이 충분하고 현재 "돈이 몰리는" 종목을 선정합니다.

*   **대상**: 미국 전체 상장 주식 (NAS, NYS)
*   **추출 소스**:
    1.  **Toss 증권 랭킹**: Node/Puppeteer 기반 미국 주식 랭킹 수집
    2.  **Yahoo Finance**: Most Active, Gainer, Loser, Growth Technology 스캐너
    3.  **Naver 증권 랭킹**: Node/Puppeteer 기반 미국 주식 랭킹 수집
    4.  **Safety Tech List**: 모든 외부 소스 실패 시에도 분석이 중단되지 않도록 보장하는 안전 기본 목록
*   **최소 요건**: 제한 없음 (초고수익 동전주 전략 반영으로 $5 최소 가격 필터 전면 제거)

> [!IMPORTANT]
> **사용자 관심종목은 공용 씨앗 목록에 직접 합치지 않습니다.** 공용 시장 스캔과 사용자 소유권을 분리하고, 관심종목 중 공용 결과에 없는 티커만 별도 분석 캐시에 저장합니다. `/scanner/latest`, 자동매매, 사용자 레이더 같은 소비 경로는 현재 로그인 사용자의 `WatchList.user_id`를 확인한 뒤에만 해당 신호를 다시 결합해야 합니다. 상세 계약은 `docs/SCANNER_DATA_FLOW.md`를 따릅니다.

### 2단계: 기술적 여과 (Stage 1: Technical Filtering)
선정된 씨앗 종목들을 15분봉(15m) 기준으로 대량 스캔하여 추세를 확인합니다.

| 필터 항목 | 기준 | 가중치 | 설명 |
| :--- | :--- | :--- | :--- |
| **Gap UP** | +3.0% 이상 | 30점 | 시가에 강한 매수세가 들어왔는가? |
| **Gap UP (약)** | +1.5% 이상 | 15점 | 소폭 갭 상승 |
| **RVOL** | 2.0배 이상 | 30점 | 최근 20개 캔들 평균 대비 거래량 폭발 |
| **RVOL (약)** | 1.2배 이상 | 15점 | 거래량 소폭 증가 |
| **New High (5일)** | -1.5% 이내 | 20점 | 최근 5일 최고가 근접 |
| **RS** | QQQ 대비 강세 | 10점 | 나스닥 지수보다 강한 주도주인가? |
| **Trend** | 9 EMA > 20 EMA | 10점 | 단기 이동평균선이 정배열 상태인가? |
| **Pre-market 갭** ⭐ | +5% 이상 | 20점 | 장 시작 전 강한 수급 신호 |
| **52주 신고가 돌파** ⭐ | 52주 고가의 98% 이상 | 25점 | 저항 없는 초강세 돌파 구간 |
| **모멘텀 연속성** ⭐ | 3봉 연속 상승 + 거래량 증가 | 15점 | 매수세 지속성 확인 |

> [!IMPORTANT]
> **통과 기준**: 합산 점수 **30점 이상** 또는 RVOL **3.0배 이상**인 종목만 Stage 2 정밀 분석으로 넘어갑니다.

> [!NOTE]
> ⭐ 표시 항목은 **v2.0 신규 추가** 지표입니다. 세 항목이 동시 충족 시 **시너지 보너스 +10점**이 추가 부여됩니다.

### 3단계: 정밀 검증 및 촉매제 (Stage 2: Precision & Catalyst)
가장 유망한 상위 25개 종목을 1분봉(1m)으로 정밀 분석하고 외부 호재를 확인합니다.

1.  **VWAP 돌파**: 주가가 당일 거래량 가중 평균가(VWAP) 위에 머물러 있는가? (+10점)
2.  **Catalyst (뉴스 🔥)**: `Earnings`, `FDA`, `Contract` 등 핵심 키워드가 포함된 호재성 뉴스가 있는가? (+10점)
3.  **Fakeout 감지**: 윗꼬리 비율(Wick Ratio)이 30% 이하인가? (고점 설거지 방지)
    *   **Low Risk**: 윗꼬리 30% 미만 (+10점)
    *   **High Risk**: 윗꼬리 50% 이상 (-20점)
4.  **거래대금(Dollar Volume) 필터**:
    *   당일 누적 거래대금이 최소 **$1,000,000(약 14억원) 이상**인 주도주만 분석합니다. 유동성이 부족한 잡주는 자동 필터링됩니다.
5.  **ATR(Average True Range) 기반 동적 변동성 라인**:
    *   해당 종목의 최근 14개 캔들의 **변동성 폭(ATR)의 1.5배~2배**를 실시간 손절/익절선으로 설정합니다.
6.  **시장 감정 필터 (QQQ 기반)**:
    *   BEARISH 장: -30점 (하락장 매수 억제)
    *   BULLISH 장: +5점 (상승장 가점)

## 🎯 4단계: 스윙 예측 파이프라인 (Daily Swing Predictor)
15분봉/1분봉 단타 스캐너와는 완전히 분리된 투트랙(Two-track) 알고리즘으로, 당장의 급등이 아닌 **"내일 이후 터질 잠재적 주도주"**를 120일 누적 일봉 데이터 기반으로 선별합니다.

| 분석 지표 | 원리 및 목적 | 가중치 |
| :--- | :--- | :--- |
| **VCP 진폭 수축** | 가격 진폭이 25% → 12% → 5% 등 우측으로 갈수록 점차 축소되며 조여지는 마크 미너비니의 VCP 패턴 감지. | 25점 |
| **VUD 거래량 극감** | 당일 거래량이 최근 20일 평균 대비 극도로 메마른 상태(Volume Dry-up). 매도세의 씨가 말랐음을 의미. | 25점 |
| **OBV 세력 매집** | 가격은 횡보/하락 중이나 OBV 지표는 우상향 다이버전스를 보일 때. (세력의 은밀한 매집 흔적) | 25점 |
| **BB 스퀴즈 대압착** | 볼린저 밴드 폭이 최근 120일 중 가장 좁은 수준(0% 근접)으로 수축하여 응축된 에너지가 폭발하기 직전 상태. | 25점 |

> [!NOTE]
> 스윙 파이프라인은 상승 추세(현재가 > EMA50)를 중시합니다. 역배열 종목은 기본적으로 **-35점의 페널티**를 받아 필터링되며, 하락장에서는 매우 소수의 종목만 통과할 수 있도록 엄격히 통제됩니다.

> [!NOTE]
> 스윙 예측은 관심종목을 결합하지 않는 인증된 공용 시장 기능입니다. 모든 사용자가 `GLOBAL_SWING_POOL`의 동일한 시장 후보를 조회하며 API 응답의 `scope`는 `global`입니다. 사용자별 관심종목 재결합은 `/scanner/latest`, 자동매매, `/account/balance` 레이더에서만 수행합니다.

## 📈 최종 시그널 판정

모든 점수를 합산하여 최종 등급을 결정하며, 대시보드에 실시간 반영됩니다.

*   **STRONG BUY (상승장 85점↑ / 약세장 95점↑)**: 모든 조건이 완벽한 대장주. **자동 매수 후보 1순위.**
*   **BUY (상승장 65점~84점 / 약세장 75점~94점)**: 추세는 좋으나 일부 재료나 지표가 부족한 상태.
*   **WATCH (상승장 40점~64점 / 약세장 50점~74점)**: 관심 종목 등록 후 돌파를 기다려야 하는 상태.
*   **NEUTRAL (상승장 40점 미만 / 약세장 50점 미만)**: 분석 대상에서 제외 (매도 대상 강세 시그널 붕괴 영역).

## 🔥 v2.0 신규 지표 상세 설명

### Pre-market 갭 감지 (장 전 급등 조기 포착)
- **원리**: 미국 정규장(09:30~16:00 ET) 이전 Pre-market 시간대에 5% 이상 갭이 발생한 종목은 장 시작 후 추가 급등 가능성이 높습니다.
- **구현**: `yfinance prePost=True` 옵션으로 Pre-market/After-hours 데이터 포함 다운로드, 전전 거래일 종가 대비 현재 Pre-market 가격 비교
- **적용 시간**: 장 시작 전(09:00~09:30 ET) 스캔 사이클에서 특히 유효

### 52주 신고가 돌파 (저항 없는 초강세 구간)
- **원리**: 52주 최고가를 돌파한 종목은 위에 매도 압력이 없어 추가 상승 탄력이 극대화됩니다.
- **기준**: 현재가 ≥ 52주 최고가의 98% (근접 또는 돌파)
- **가중치**: +25점 (단일 지표 중 최고 가중치)

### 연속 상승 모멘텀 (Momentum Continuation)
- **원리**: 직전 3개 봉이 연속으로 종가 상승하면서 거래량도 증가할 경우, 매수세가 단발성이 아닌 지속적임을 확인합니다.
- **조건**: `Close[-1] > Close[-2] > Close[-3]` AND `Volume[-1] > Volume[-2] > Volume[-3]` (모두 15분봉 기준)
- **효과**: Fakeout(가짜 돌파) 확률 대폭 감소

### 세 지표 동시 충족 시 시너지 보너스
```
Pre-market 갭 ≥ 5%
+ 52주 신고가 근접/돌파
+ 3봉 연속 상승 + 거래량 증가
= +10점 시너지 보너스 자동 부여
```
이 세 조건의 동시 충족은 시장에서 매우 드물며, 해당 종목은 **기관 매집 또는 세력의 계획적 매수**일 가능성이 매우 높습니다.

## 🛠️ 유지보수 및 업데이트 규칙
- 시장 상황에 따라 RVOL 기준값(2.0x)은 조정될 수 있습니다.
- 뉴스 키워드는 `scanner.py`의 `CATALYST_KEYWORDS` 상수를 통해 관리합니다.
- 스캐너 캐시는 **10분 주기**로 자동 갱신되며, 자동매매 체결은 **1분 주기**로 캐시를 참조합니다.
- 급등 추격매수 차단: 캐시 시그널 대비 **+20% 이상 가격 상승 시** 매수 자동 차단.
- 공용 시드·관심종목·사용자별 재결합 흐름을 변경하면 `docs/SCANNER_DATA_FLOW.md`와 소비 경로 회귀 테스트를 같은 작업에서 갱신합니다.
