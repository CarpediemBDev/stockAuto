# 🗺️ StockAuto v2.0 천하통일 전략 통합 마스터 맵 (Strategy Map)

본 문서는 StockAuto 시스템에 구현된 **모든 개별 트레이딩 전략(14대 전략)과 11대 핵심 대가 기법들**이 소스코드의 어느 파일에 구현되어 있고, 어떤 시장 환경(Regime)에서 동작하며, 디폴트 백테스트 환경과 핵심 매매 로직은 어떻게 설계되어 있는지 한눈에 추적하도록 요약한 **공식 전략 마스터 테이블**입니다.

---

## 🏆 전략 등급 및 활성 장세 아이콘 표준 정의 (Tiers & Regimes)

마스터 테이블의 모든 전략은 **성능/위상별 등급(Tier)**과 **활성 장세(Regime)별 역할**에 따라 다음과 같이 공식 시각적 배지(이모지)로 완벽히 일치화되어 관리됩니다.

### 1. 전략 등급 배지 (Tiers)
* **🥇 금메달 (Gold Tier)** : 11대 핵심 기법이 레짐별로 조율된 종합 1위 완전체 메인 전략 (complex)
* **🥈 은메달 (Silver Tier)** : 노이즈를 완전 걷어낸 직관적인 시니어 맞춤형 2위 단순화 전략 (senior_simple)
* **🥉 동메달 (Bronze Tier)** : StockAuto의 역사적 기초 뼈대를 구축한 기본 3위 클래식 전략 (strategy_a)
* **🧪 실험중 (Sandbox Tier)** : 파라미터 미세 조정 및 비중 분기를 검증하는 활성 연구/테스트 단계 전략 (strategy_b, exploded_c)

### 2. 활성 장세별 기능 배지 (Regimes)
* **🔥 상승장 (BULLISH)** : 공격형 모멘텀 돌파 및 추격 매수 기동 (Fire)
* **🛡️ 하락장 (BEARISH)** : 방어형 바닥 매집 및 과매도 극점 지지 반등 기동 (Shield)
* **⚡ 횡보장 (NEUTRAL)** : 횡보 압축 구간 에너지 분출 및 레인지 스윙 기동 (Squeeze)
* **📡 모든장세 (ALL)** : 거시 시장 흐름을 감시하여 장세에 맞춰 매매 모드를 조율 및 다이내믹 변환 (Orchestrator)

## ⚙️ 2. 4대 매매 라이프사이클 작동 시나리오 (Strategy C 종합)

1.  **[1단계: 종목발굴]**
    *   **상승장:** 시게루의 재무 필터 통과 ➡️ 쿨라매기의 52주 고가 근접 ➡️ 전수조사 스코어링 80점 이상STRONG_BUY 픽.
    *   **하락/횡보장:** 시게루의 재무 필터 ➡️ 차트픽의 OBV 세력 매집 ➡️ MACD 다이버전스 ➡️ 추세선 돌파 리테스트 ➡️ 스코어링 90점 이상 보수적 픽.
2.  **[2단계: 진입타점]**
    *   **상승장:** 토비 크라벨의 5분봉 ORB 돌파 ➡️ 비트고수 2200%의 Pre-market 갭 돌파 시점에 API 주문 예약 전송.
    *   **하락/횡보장:** 비트고수의 RSI 볼밴 하단 회귀 ➡️ 피보나치 38.2% 되돌림 ➡️ 20 EMA 성배 눌림 아래꼬리 터치 시 주문 전송.
3.  **[3단계: 자금관리]**
    *   **상승장:** 후지모토 시게루의 1:2:6 법칙 적용. 최초 15%(정찰병) 매수 ➡️ 15분봉 안착 시 35% 불타기 ➡️ 고가 돌파 시 50% 풀배팅.
    *   **하락/횡보장:** 피라미딩(불타기) 절대 금지. 1회성 보수적 매수(비중 30% 축소 제한)만 실행.
4.  **[4단계: 매도/탈출]**
    *   **스마트 이절:** 분봉상 비트고수의 RSI 다이버전스(상승 피로)와 MACD 데드크로스가 동시 포착 시 즉시 시장가 이절.
    *   **안전 손절:** ATR 기반의 동적 손절선 또는 직전 저점 지지선 붕괴 시 예외 없는 시장가 전량 칼손절.

## 🥇 3. 격리형 2슬롯 모듈러 코어 자본 격리 아키텍처 (Modular Capital Isolation)

단일 증권 계좌 내 자본의 안전을 지키고, 상이한 두 코어 알고리즘 간의 자금 간섭을 원천 차단하기 위해 **수학적인 격리형 2슬롯 자본 배분 구조**를 가동합니다.

```
                      [ 💰 단일 증권 계좌 총 자산 (100%) ]
                                       ⬇
              ---------------------------------------------------
             ⬇ (수학적 50:50 격리 장벽 - 장세 무관 영구 적용)  ⬇
     【 ⚡ EP (에피소딕 피벗) 슬롯 】                 【 👑 RS V2 (레짐스위칭) 슬롯 】
     - 계좌 총 자산의 50% 지분 소유                   - 계좌 총 자산의 50% 지분 소유
     - 개별 변동성 돌파 및 강세 수급 공략            - 상승 대국면 추종 및 불타기 피라미딩
```

### 🛡️ 자본 격리 및 보존의 3대 핵심 규칙 (Safety Rules)

1. **상하 장벽 격리 보존 법칙**:
   * 상승장(BULLISH)이 되어 두 전략 슬롯의 가동 한도가 100% 활짝 열리더라도, **두 슬롯 간의 자본 격리 장벽(Isolation Barrier)은 절대 허물어지지 않습니다.**
   * 만약 EP 조건에 맞는 돌파 종목이 없어 EP 슬롯의 자금이 전부 놀고 있더라도, **RS 슬롯은 EP의 노는 자금 50%를 뺏어오거나 빌려 쓸 수 없습니다.**
   * 이 장벽 덕분에 RS 전략이 아무리 좋은 상승세라도 계좌의 최대 50% 자금 한도 내에서만 가동되므로, 갑작스러운 시장 악재 시 100% 몰빵으로 인한 계좌 파산 리스크를 철저히 예방합니다.

2. **상승장(BULLISH) 시너지 가동**:
   * QQQ 지수가 강세일 때는 두 슬롯의 인출/가동 한도가 모두 100%(weight = 1.0)로 오픈되어 **EP 종목 50% + RS 종목 50% = 계좌의 100% 자본을 꽉 채워 시장에 투자**할 수 있습니다.

3. **하락/횡보장(BEARISH/NEUTRAL) 현금 대피 가드**:
   * 하락장 진입 시 RS 슬롯의 가동 한도는 **0%로 완전 잠금(Blocked)**됩니다. 
   * 이에 따라 RS 슬롯 소유의 50% 자본은 주식을 단 1원도 사지 않고 **100% 안전하게 현금(예수금) 형태로 금고에 긴급 피난 보존**되어 계좌의 강력한 방어벽이 됩니다.
